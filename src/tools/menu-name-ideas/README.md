# 메뉴명 아이디어 (메뉴명 추천)

AI로 업종·메뉴 종류·키워드·톤에 맞는 **메뉴명 후보**를 추천하는 툴.

## 입력
- **업종**: 텍스트. 예: 카페, 한식, 베이커리
- **메뉴 종류**: 음료, 메인, 디저트, 사이드, 기타
- **키워드**(선택): 예: 달달한, 건강한
- **톤**(선택): 감성, 심플, 재치, 프리미엄, 친근
- **개수**: 5~15

## 출력
- 메뉴명 후보 리스트 (한 줄씩)
- 사용된 AI 프로바이더명 (표시용)
- 각 항목 복사 버튼

## API
- `POST /api/tools/menu-ideas` — body: category, menuType, keywords?, tone?, count
- LLM: Gemini → Groq → OpenRouter 순차 폴백

## 프롬프트
- 시스템: 한국어, 메뉴명만 한 줄씩, 개수 준수, 상표 유사 표현 금지
- 사용자: 업종·메뉴종류·키워드·톤·개수 조합
