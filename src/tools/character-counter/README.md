# 글자수 카운터

## 툴 개요
텍스트를 입력하면 글자수, 바이트수, 공백 포함/제외 글자수를 즉시 계산합니다.
SNS 홍보 문자, 카카오톡 메시지, 네이버 플레이스 소개글 작성 시 유용합니다.

## 입력 항목
- 텍스트 (textarea)

## 출력 항목
- 전체 글자수 (공백 포함)
- 글자수 (공백 제외)
- 줄 수
- 바이트 수 (한글 3바이트, 영문/숫자 1바이트)
- 단어 수

## 계산 로직
- 전체 글자수: `text.length`
- 공백 제외 글자수: `text.replace(/\s/g, '').length`
- 줄 수: `text.split('\n').length`
- 바이트 수: 각 문자를 UTF-8 기준으로 계산 (한글 3바이트, ASCII 1바이트, 기타 2바이트)
- 단어 수: 공백/줄바꿈 기준 단어 분리

## UI 요구사항
- 큰 textarea (최소 6줄)
- 실시간 계산
- 초기화 버튼
- 자주 쓰이는 제한 글자수 표시 (카카오 1000자, 인스타 2200자, SMS 90자)
