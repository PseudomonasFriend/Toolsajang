# 평당 임대료 계산기

## 툴 개요
매장 면적(평 또는 m²)과 월세를 입력하면 평당 임대료를 계산합니다.
여러 매장 후보를 비교할 때, 실제 임대료 부담을 정확히 비교할 수 있습니다.

## 입력 항목
- 면적 (평 또는 m², 선택)
- 보증금
- 월세
- 관리비 (선택)

## 출력 항목
- 평 환산 면적 (m²로 입력 시)
- 평당 월 임대료 (월세 기준)
- 평당 실질 임대료 (월세+관리비 기준)
- 연 환산 임대료 총액
- 권리금 회수 기간 (권리금 입력 시, 월이익 기준)

## 계산 로직
- 1평 = 3.305785m²
- 평당 월세 = 월세 / 평수
- 평당 실질 임대료 = (월세 + 관리비) / 평수
- 연 임대료 = (월세 + 관리비) × 12

## 적정 임대료 기준
- 매출의 10% 이하가 적정 (일반적 기준)
- 업종별로 다름: 식음료 7~12%, 의류 10~15%
