# 재고 회전율 계산기

## 툴 개요
매출원가와 평균 재고금액을 입력하면 재고 회전율과 재고 보유 일수를 계산합니다.
재고가 너무 많아 창고비 낭비, 또는 재고 부족으로 판매 기회 손실을 방지하기 위한 툴.

## 입력 항목
- 기간 (1개월/분기/반기/1년)
- 매출원가 (해당 기간 원가 합계)
- 기초 재고금액 (기간 시작 시점 재고)
- 기말 재고금액 (기간 종료 시점 재고)

## 출력 항목
- 평균 재고금액
- 재고 회전율 (연환산)
- 재고 보유 일수
- 평가 메시지 (회전율에 따른 간단한 진단)

## 계산 로직
- 평균 재고 = (기초 재고 + 기말 재고) / 2
- 연간 매출원가 = 매출원가 × (365 / 기간 일수)
- 재고 회전율 = 연간 매출원가 / 평균 재고
- 재고 보유 일수 = 365 / 재고 회전율

## 업종별 적정 회전율 기준
- 식음료/제과: 30회 이상 (보유일수 12일 이하)
- 의류/잡화: 4~8회 (보유일수 45~90일)
- 일반 소매: 6~12회 (보유일수 30~60일)
- 도매: 3~6회 (보유일수 60~120일)
