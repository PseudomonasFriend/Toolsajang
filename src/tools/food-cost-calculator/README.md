# 원가율 계산기 (식음료 F&B)

## 툴 개요
메뉴별 식재료 원가와 판매가를 입력하면 원가율, 적정 판매가, 목표 원가율 기반 권장 가격을 계산합니다.
음식점, 카페, 식품 소매업 사장님을 위한 필수 도구입니다.

## 입력 항목
- 메뉴 이름 (선택)
- 식재료 원가 합계
- 판매가 (현재 가격)
- 목표 원가율 (%) — 기본값: 30%

## 출력 항목
- 현재 원가율 (%)
- 원가 이익 (판매가 - 식재료 원가)
- 목표 원가율 기준 권장 판매가
- 진단 메시지

## 계산 로직
- 원가율 = (식재료 원가 / 판매가) × 100
- 권장 판매가 = 식재료 원가 / (목표 원가율 / 100)
- 원가 이익 = 판매가 - 식재료 원가

## 업종별 적정 원가율
- 일반 식당: 30~35%
- 카페·음료: 20~30%
- 패스트푸드: 25~35%
- 베이커리·제과: 35~45%
